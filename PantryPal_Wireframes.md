## Design System Elements

### Color Palette
- **Primary**: #4CAF50 (Green - fresh, healthy)
- **Secondary**: #FF9800 (Orange - warm, appetizing)
- **Accent**: #2196F3 (Blue - trust, technology)
- **Background**: #FAFAFA (Light gray)
- **Surface**: #FFFFFF (White)
- **Text**: #212121 (Dark gray)
- **Error**: #F44336 (Red)
- **Success**: #4CAF50 (Green)
- **Warning**: #FF9800 (Orange)

### Typography
- **Headings**: Inter, Bold, 24px/20px/18px
- **Body**: Inter, Regular, 16px
- **Captions**: Inter, Medium, 14px
- **Buttons**: Inter, Medium, 16px

### Iconography
- **Food items**: Custom food icons
- **Actions**: Material Design icons
- **Navigation**: Simple, recognizable icons
- **Status**: Color-coded indicators

### Spacing System
- **XS**: 4px
- **S**: 8px
- **M**: 16px
- **L**: 24px
- **XL**: 32px
- **XXL**: 48px

### Component Library
- **Cards**: Rounded corners (8px), subtle shadows
- **Buttons**: Primary (filled), Secondary (outlined), Text
- **Inputs**: Floating labels, validation states
- **Navigation**: Bottom tabs (mobile), top navigation (web)
- **Modals**: Slide-up (mobile), centered (web)

This wireframe system provides a comprehensive visual foundation for implementing the PantryPal app across both mobile and web platforms.
